Welcome & Thank You Quarto

news
Author

Athul Sudheesh

Published

April 7, 2022

I came to know about Quarto a couple of days ago, and since then, I have been in love with it. This blog is built using Quarto Publishing, and I found it to be very straightforward and easy. Previously this blog was built using Franklin.jl and before that using Jekyll. One of the reasons I moved from Franklin to Quarto was Quarto’s language-agnostic approach and advanced layout setting.

I hope to use this space to share my thoughts and notes on quantitative psychology, theoretical statistics, and machine learning. My goal is to use these github pages for articles aimed at technical audience and use my main website for general public. There are two reasons for me to choose github pages/quarto combo over my wordpress website:

  • I found it hard to get syntax highlight support for Julia language. Most of my computational work are done in Julia.
  • I want these notes to be eternally accessible.
